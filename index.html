Aprendendo Sobre Tags

(Voltar)

Durante o bootcamp da DIO, aprendi várias tags HTML que ajudam a organizar o conteúdo das páginas. Os títulos vão do h1 ao h6, do mais importante ao menos. Os textos comuns ficam dentro de p, e se quiser destacar algo, dá pra usar mark (marca-texto), small (texto menor), i (itálico), u (sublinhado) e strong (negrito).

Também conheci as listas: ol para listas numeradas, ul para listas com bolinhas e li para os itens. Os links são feitos com a e a hr insere uma linha horizontal. Já sub e sup deixam o texto embaixo ou em cima da linha, e blockquote serve para colocar citações destacadas.

Além dessas, vi algumas tags extras: del, que risca o texto; e abbr, que mostra o significado de siglas quando passamos o mouse.

Com essas tags básicas, já dá pra montar páginas bem organizadas e sem mistério.

Aprendendo Sobre Formulários

(Voltar)

Formulários são áreas de uma página onde o usuário pode preencher informações e enviá-las para o servidor. Eles são super comuns em cadastros, buscas, comentários e qualquer interação que envolva entrada de dados.

Quando o usuário preenche os campos e clica no botão de envio, o navegador envia essas informações para o servidor, que processa tudo e pode salvar no banco de dados, retornar um resultado ou fazer qualquer outra ação.

As principais tags utilizadas são:

  • form: Define o início e o fim de um formulário. Tudo que for para envio de dados fica dentro dela.
  • input: Usada para inserir informações. Pode ter vários tipos: texto, senha, email, número, entre outros.
  • checkbox e radio: Permitem selecionar opções. Checkbox aceita várias seleções, radio permite só uma.
  • button: Botão que, geralmente, envia o formulário. Pode ter o tipo submit, reset ou button.
  • select: Cria um menu suspenso com várias opções para o usuário escolher.
  • textarea: Campo para textos maiores, como mensagens ou descrições.

Aprendendo Sobre Tabelas

(Voltar)

Tabelas são usadas no HTML para organizar informações em linhas e colunas, como se fosse uma planilha. Elas ajudam a exibir dados de forma estruturada e fácil de visualizar.

Principais tags de tabelas são:

  • table: Cria a tabela.
  • tr: Define uma linha dentro da tabela.
  • td: Cria uma célula de dado (conteúdo normal) dentro de uma linha.
  • th: Cria uma célula de cabeçalho, normalmente aparece em negrito e centralizada.
  • thead: Agrupa os cabeçalhos da tabela.
  • tbody: Agrupa o conteúdo principal da tabela.
  • tfoot: Define o rodapé da tabela, ótimo para totais ou informações finais.

Essas tags juntas permitem criar tabelas organizadas, com cabeçalhos, corpo e rodapé bem definidos. Além disso, dá pra estilizar tudo isso e deixar a tabela mais bonita e responsiva.

Importância do HTML Semântico e da Acessibilidade

(Voltar)

Um HTML semântico significa usar as tags certas para cada tipo de conteúdo. Ou seja, não é só colocar tudo dentro de div e span, mas sim usar tags específicas, citadas anteriormente, que indicam o papel daquele conteúdo na página. Isso ajuda tanto os navegadores quanto ferramentas de acessibilidade (como leitores de tela) a entenderem melhor a estrutura e a função de cada parte do site.

Além disso, pensar em acessibilidade é garantir que qualquer pessoa, independente de limitações físicas, visuais ou cognitivas, consiga navegar e interagir com seu conteúdo. Isso envolve desde organizar bem as informações até usar atributos como alt nas imagens e aria-label em botões.

No fim, um HTML bem estruturado e acessível melhora a experiência de todo mundo, facilita para os mecanismos de busca (SEO) e deixa seu site mais inclusivo e profissional.
